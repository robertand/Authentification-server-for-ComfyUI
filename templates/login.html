ComfyUI Auth Server v0.4.0

Advanced management and authentication system for multiple ComfyUI instances

Key Features and Advantages:

  • Multi-User Architecture - Simultaneous management of multiple users with separate instances
  • Intelligent Load Balancing - Automatic workload distribution between ComfyUI instances
  • Enhanced Security - Rate limiting, bcrypt password hashing, customizable session timeouts
  • Advanced Admin Interface - Real-time monitoring, session management, full control
  • Integrated Chat System - Direct communication between users and administrators
  • Personalized Workflow Browser - Each user has their own workspace for workflows
  • Session Management - Customizable timeout, expiry notifications, session renewal
  • Performance Monitoring - Real-time instance status, maximum capacity alerts
  • Backup & Restore - Automatic configuration saving, easy recovery
  • Scalability - Easy addition of new users and resources
  • Nginx Reverse Proxy Support - Full support for nginx with Basic Authentication

Security Features:

  • Authentication with hashed passwords (bcrypt)
  • Protection against brute-force attacks
  • Configurable session timeout per user
  • Temporary user blocking for security
  • Suspicious IP monitoring
  • Nginx Basic Auth support (per user or global)

For Administrators:

  • Real-time statistics dashboard
  • Complete user management
  • Flexible instance configuration
  • Chat system with users
  • Active session control
  • Workflow directory configuration
  • Nginx authentication configuration per user

Main Benefits:

Resource Optimization - Balanced distribution of computing load
User Isolation - Each user works in a secure space
Easy Management - Single interface for all instances
Scalability - Always add new users and resources
Nginx Integration - Works seamlessly with nginx reverse proxies
